# Gerenciador de Pontos Corridos

Acesse a aplicação em: [https://asolera.github.io/gerenciador-pontos-corridos/](https://asolera.github.io/gerenciador-pontos-corridos/)

## Rodando Localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse a aplicação no navegador (geralmente em `http://localhost:3000`).
